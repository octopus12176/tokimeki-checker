# paths: src/\*.html

## HTML ガイドライン

### 1. ベストプラクティス

- セマンティックなタグを適切に使う（header, main, section など）
- インライン JavaScript を使用しない（セキュリティ上の理由）
- script タグは原則 defer を利用する
- 不要な入れ子構造を避ける

### 2. コメント

- 日本語で簡潔に書く
- 実装理由や注意点を中心とする

### 3. 禁止事項

- innerHTML へユーザー入力を直接埋め込む例を推奨しない
- 不必要なタグ変更（div → section など）を勝手に行わない
